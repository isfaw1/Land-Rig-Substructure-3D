## 📋 What is Land-Rig-Substructure-3D?

This application lets you view a detailed 3D model of a land rig substructure. It shows the main parts of the rig’s base in clear, interactive 3D graphics. The plot runs through your web browser, making it simple to use without special software.

You can rotate, zoom, and explore the model to understand how land rig substructures look and operate. This tool suits users interested in drilling operations, mechanical engineering, and petroleum engineering. It helps visualize complex mechanical parts found in rig bases.

## ⚙️ How It Works Behind the Scenes

This application uses a set of 3D models stored in OBJ and MTL file formats. These formats describe the geometry and materials of the land rig substructure.

The site loads these models into a 3D viewer built with web technology. This viewer displays the mesh and texture using your browser’s graphics capabilities. It does not require special software or programming knowledge.
